**Last Updated**: 2026-03-29

## Research Team Workflow

```
Request → Research → Report → Review → Update Docs
```

| Phase | Owner | Output |
|-------|-------|--------|
| Request | Team Lead | Research query |
| Research | Research Agent | Draft report |
| Report | Research Agent | MD report in research/ |
| Review | Team Lead | Feedback |
| Update Docs | Research Agent | Update coordination docs |

## Research Output Format

All research outputs follow this structure:

```
research/{agent-type}/{topic}-{type}-{date}.md
```

| Type | Purpose | Template |
|------|---------|----------|
| report | Detailed findings | Full analysis |
| summary | Quick overview | 1 page max |
| action-items | Recommendations | Checklist |
